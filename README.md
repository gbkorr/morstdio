# morstdio
This is the repo for my [morse code i/o](https://gbkorr.github.io/r-bites/morstdio/morstdio.html), a shell script that lets you send and receive morse code from a bash shell using the boot button and user led. See the link for a full guide.

It's made to run on a Luckfox Lyra board, but it should be easy to adapt to whatever else.
